# 基于Ajax开发的Weibo应用
    设计了后端api端口，异步处理用户请求，无需刷新网页即可实现添加、删除功能
    采用MVC架构，程序可重构性强，易于维护
    减少带宽和服务器网络负载，提高页面访问速度
